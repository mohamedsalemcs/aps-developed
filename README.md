# دليل استخدام لوحة تحكم موقع APS

دليل مختصر يوضّح كيفية إدارة محتوى الموقع من لوحة التحكم.

## روابط النسخة التجريبية (PythonAnywhere)
| | الرابط |
|---|---|
| **الموقع** | https://mohamedsalemcs.pythonanywhere.com/ |
| **لوحة التحكم** | https://mohamedsalemcs.pythonanywhere.com/cms/login/ |

**بيانات الدخول إلى لوحة التحكم:**
- اسم المستخدم: **`aps_admin`**
- كلمة المرور: **`Admin12345`**

> يُنصح بتغيير كلمة المرور بعد أول تسجيل دخول من صفحة **«الملف الشخصي»**.

## 1) تسجيل الدخول
- افتح رابط لوحة التحكم أعلاه، وأدخل اسم المستخدم وكلمة المرور.
- يمكنك تبديل لغة اللوحة من زر **«عربي / English»** أعلى الصفحة.

## 2) ما الذي يمكنك تعديله، ومن أين
لكل صفحة في الموقع صفحة تحرير مقابلة في القائمة الجانبية:

| العنصر في القائمة | المحتوى القابل للتعديل |
|---|---|
| **الصفحة الرئيسية** | القسم الرئيسي (العنوان والنص والأزرار)، نبذة عن APS، بطاقات الأقسام (صورة وعنوان ووصف)، الشركاء، التواصل |
| **عن الشركة** | البانر، من نحن، الرؤية والرسالة والقيم، مبادئ العمل |
| **الأقسام (الخمسة)** | لكل قسم: البانر، نبذة عن القسم، القسم الرئيسي، المشاريع، بيانات التواصل |
| **الأسئلة الشائعة** | الأسئلة وإجاباتها |
| **اتصل بنا** | عناوين الصفحة ونصوصها |
| **الشركاء** | شعارات الشركاء (إضافة / استبدال / حذف) |
| **الهوية والتصميم** | الشعار (الرأس والتذييل)، الألوان، الخطوط |
| **الإعدادات** | رقم الهاتف، البريد الإلكتروني، الموقع الإلكتروني، العنوان |

## 3) كيفية الحفظ
- بعد تعديل أي حقل، اضغط زر **«حفظ»** أسفل الصفحة.
- لكل حقل نسختان: **العربية والإنجليزية** — يُرجى تحديث النسختين معًا.
- بعد الحفظ، افتح الموقع واضغط **Ctrl + Shift + R** لعرض التحديثات.

## 4) أدوات التحكم في أقسام صفحات الأقسام
داخل صفحة تحرير أي قسم، يحتوي كل قسم فرعي على خمسة أزرار:
- **⬆️ / ⬇️** لإعادة الترتيب (رفع القسم أو خفضه).
- **👁** لإظهار القسم أو إخفائه من الموقع.
- **🗑** لحذف القسم.
- **⌄** لطيّ البطاقة أو فتحها.

## 5) نقاط مهمة يُرجى مراجعتها
1. **النصوص العربية:** المحتوى المستورد من ملف العميل كان باللغة الإنجليزية، وتُرجمت نسخته العربية تلقائيًا — يُرجى **مراجعة الصياغة العربية** وتعديلها من لوحة التحكم عند الحاجة.
2. **شعارات الشركاء:** عند رفع شعار جديد، يُطبَّق عليه الإطار الموحّد **بعد الحفظ** (يظهر في المحرِّر بدون إطار إلى أن يتم الحفظ وإعادة تحميل الصفحة).

> تُحفظ جميع التعديلات في الموقع مباشرةً. وفي حال حدوث أي خطأ في صفحة ما، يتوفّر زر **«إعادة إلى الإعدادات الافتراضية»** الذي يعيد الصفحة إلى محتواها المعتمد.
